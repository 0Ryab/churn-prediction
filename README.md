# Churn Prediction

Projeto de previsão de churn de clientes bancários usando Machine Learning.

## Objetivo
Prever a probabilidade de um cliente encerrar sua conta, permitindo
ações proativas de retenção.

## Estrutura
- `data/` — dados brutos e processados
- `notebooks/` — exploração e experimentos
- `src/` — código modular e reutilizável
- `models/` — modelos treinados salvos
- `reports/` — visualizações e resultados

## Setup
```bash
python -m venv venv
venv\Scripts\activate
pip install -r requirements.txt
```
